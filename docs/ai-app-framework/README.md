# AI App Creation Prep Pack

This folder is the app-building companion to the website framework pack.

It is meant to prepare you before starting a real application build so you do not have to rediscover the toolchain, dependencies, accounts, or AI workflow every time.

## What This Pack Covers

- machine setup
- accounts and services
- core app stack
- optional feature-specific dependencies
- app architecture defaults
- AI skills and plugin workflow
- installation commands
- decision matrix for auth, database, email, uploads, payments, and AI
