# AI Website Framework Pack

This folder is a reusable operating system for building premium brochure, marketing, portfolio, and showcase websites with an AI assistant.

It is designed to take you from:
- point A: no website, no structure, loose brief
- point B: a real deployed site with working interactions, mobile QA, real contact flow, GitHub, and Vercel

## Recommended Use Order

1. Fill out [SITE_BRIEF_TEMPLATE.md](C:/Users/Admin/Desktop/izcqlonotsait/docs/ai-website-framework/SITE_BRIEF_TEMPLATE.md)
2. Give the completed brief plus [MASTER_PROMPT.md](C:/Users/Admin/Desktop/izcqlonotsait/docs/ai-website-framework/MASTER_PROMPT.md) to the AI
3. Use [WORKFLOW_AND_GATES.md](C:/Users/Admin/Desktop/izcqlonotsait/docs/ai-website-framework/WORKFLOW_AND_GATES.md) as the process and QA operating system
4. Use [FULL_BUILD_PLAYBOOK.md](C:/Users/Admin/Desktop/izcqlonotsait/docs/ai-website-framework/FULL_BUILD_PLAYBOOK.md) for the full end-to-end build path from empty repo to live website
5. Use [TOOLS_SKILLS_AND_PLUGINS.md](C:/Users/Admin/Desktop/izcqlonotsait/docs/ai-website-framework/TOOLS_SKILLS_AND_PLUGINS.md) to make sure the software stack, AI capabilities, plugins, testing tools, and deployment tools are all present

## What This Pack Covers

- project briefing

- modular implementation architecture
- content-system design

- interaction QA
- real asset replacement
- form integration
- GitHub setup
- Vercel deployment
- post-launch edit workflow

## Output Philosophy

This pack is not meant to produce generic "AI website" output.

It is optimized for websites that:
- feel designed, not generated
- use strong hierarchy and controlled motion
- survive content changes cleanly
- work on real phones, not just desktop emulation
- are maintainable after launch

## Scope

This framework is strongest for:
- premium portfolio sites
- brochure sites
- service business websites
- marketing websites
- small multi-page brand sites

It can still be adapted to:
- SaaS marketing sites
- creative studio websites
- consultant and agency sites
- premium e-commerce landing experiences
